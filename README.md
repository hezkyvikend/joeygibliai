# joeygibliai

Resume and portfolio website built with Vite, React, TypeScript, Tailwind, and shadcn-style UI components.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Edit content

- Profile info: `src/config/personal.ts`
- Experience: `src/data/experience.ts`
- Projects: `src/data/projects.ts`

## Assets

- Avatar: `src/assets/avatar.png`
- CV PDF: `public/Joseph_Gibli_CV.pdf`
